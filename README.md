# cm1
Cm1 (C minus 1 or C - 1) is a subset of C programming language that compiles to bytecode very quickly. It compiles faster than Tiny C Compiler. All Cm1 codes are compilable by GCC and Clang but the reverse is not true. This means you can iterate, test, debug, edit codes and recompile very quickly during development and just compile the Cm1 codes as C with no modification when making a release build. Cm1 can also be used as a scripting language for shell scripts or video games.

My goal with this programming language is to be a target language of my other programming language called Cp1 (cp1-lang.org) so that Cp1 can become a true compiler instead of just a transpiler.
